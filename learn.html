Legacy API

Step 1: Registration

To get an access to our portal please drop a email to us. development@connxai.com . Our team will contact you ASAP. Once you register with us please follow these below steps to interact with our API's.

Integration With Our API:

Click on products shown in the Navbar and select any end point from dropdown.it will take us to swagger page.

Step 2: Login
  1. To get the required data we need to complete login and authorization.
  2. Before generating the token, we need to login by clicking Django login button
  3. To complete authorization, we will be needed Token generated w.r.t user

Step 3: Collect Token
  1. From the Django admin panel
  2. Directly from swagger UI
Generating token from swagger ui:
  1. In above page click on “api” post

Click on try it out

Then it will allow us to edit our login credentials.after entering login credentials click on execute.

Then it will Under response body we will be generated.copy the generated token and click on Authorize located at right top corner of the page

  1. On click one window will pop up.then paste the generated token in given space.


  2. Note:Token should be entered in format of “Token” generated token value


  3. Click on “Authorize” , If Token is valid ,session will be start. If token is not valid it will throw an error while firing an API end point.


  4. Go to swagger page .here we can able to see endpoints. Select the required end point and click on”GET”


  5. By clicking “try it out ” button it will allow us to enter the store number and click on “Execute”. API would be fired and we can able see the Json data on screen
